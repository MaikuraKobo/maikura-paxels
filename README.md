# Maikura Paxels v1.0.0

Fabric 1.21.11 向けの4種統合パクセルMODです。

## 追加アイテム
- 木のパクセル
- 石のパクセル
- 銅のパクセル
- 鉄のパクセル
- 金のパクセル
- ダイヤパクセル
- ネザライトパクセル
- 光導のパクセル

## 基本仕様
- ツルハシ・斧・シャベル・クワ対象ブロックを1本で採掘
- 通常パクセルの耐久は素材デフォルトの2倍
- 光導のパクセルはネザライトパクセルと同耐久
- 4種ツール系エンチャントに対応

## 操作
- 右クリック：樹皮剥がし / 土の道
- スニーク右クリック：耕作

## 強化
- ダイヤパクセル + ネザライト強化テンプレート + ネザライトインゴット → ネザライトパクセル
- ネザライトパクセル + ビーコン + ネザライトインゴット → 光導のパクセル

## 光導のパクセル
- メインハンド所持時にヘイスト I
- 修理不可、修繕のみ対応
- Mod Menuアイコンに専用アイコンを設定
